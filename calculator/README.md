# Calculator | ماشین حساب

یک ماشین حساب ساده با HTML, CSS و JavaScript

A simple calculator built with HTML, CSS, and JavaScript

## فایل‌ها | Files
- `index.html` – صفحه HTML  
- `style.css` – استایل  
- `calculator.js` – منطق جاوااسکریپت

---

## اجرا | How to Run
1. `index.html` را در مرورگر باز کنید  
2. برای تغییر ظاهر یا عملکرد، کد JS و CSS را ویرایش کنید

Open `index.html` in a browser  
Edit JS or CSS files to modify appearance or functionality
