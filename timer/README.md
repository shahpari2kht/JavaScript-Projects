# Timer | تایمر

تایمر ساده با HTML, CSS و JavaScript

Simple timer built with HTML, CSS, and JavaScript

## فایل‌ها | Files
- `index.html` – صفحه اصلی  
- `style.css` – استایل  
- `timer.js` – منطق تایمر

---

## اجرا | How to Run
1. `index.html` را در مرورگر باز کنید  
2. تایمر شروع به کار می‌کند و می‌توانید تنظیمات را تغییر دهید
