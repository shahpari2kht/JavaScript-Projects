# Basic-JS-Practice | تمرین‌های پایه JS

این پروژه شامل تمرین‌های پایه جاوااسکریپت است: متغیرها، توابع و شرط‌ها

This project contains basic JavaScript exercises: variables, functions, and conditionals

## فایل‌ها | Files
- `variables.js` – تمرین متغیرها  
- `functions.js` – تمرین توابع  
- `conditions.js` – تمرین شرط‌ها

---

## اجرا | How to Run
1. فایل‌های JS را با مرورگر یا Node اجرا کنید  
2. از Console مرورگر برای مشاهده نتایج استفاده کنید

Run JS files in browser or Node.js  
Use browser console to see the output
