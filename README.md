# JavaScript-Projects | پروژه‌های جاوااسکریپت

**My JavaScript Projects | پروژه‌های آموزشی و تمرینی JavaScript**  

این ریپازیتوری شامل مجموعه‌ای از پروژه‌ها و تمرین‌های مختلف با زبان **JavaScript** است.  
هدف: نمایش مهارت‌های **JavaScript، منطق برنامه‌نویسی، DOM Manipulation و توسعه وب تعاملی**.

This repository contains a collection of various **JavaScript** projects and exercises.  
Purpose: showcase **JavaScript skills, programming logic, DOM manipulation, and interactive web development**.

## راهنمای اجرا | How to Run
1. به فولدر پروژه مورد نظر بروید  
2. فایل `index.html` را در مرورگر باز کنید  
3. برای ویرایش یا تست کد، از ادیتور دلخواه استفاده کنید

Navigate to the project folder  
Open `index.html` in a web browser  
Use any code editor to modify or test the code
